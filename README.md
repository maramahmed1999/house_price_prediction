# House Price Prediction

## Project Overview

This project focuses on analyzing housing data and building a machine learning model to predict house prices based on different property features.
The main goal of the project was to practice data preprocessing, exploratory data analysis, feature engineering, and regression modeling using real-world housing data.

## Objectives

* Understand housing market data and feature relationships.
* Perform data cleaning and preprocessing.
* Analyze important factors affecting house prices.
* Apply feature engineering techniques.
* Train and evaluate regression models for price prediction.
* Improve model performance using data analysis and preprocessing techniques.

## Dataset

The dataset contains information about houses and property characteristics such as:

* Area and location
* Number of rooms
* Bathrooms
* Floors
* Furnishing status
* Parking availability
* Main road access
* Preferred area
* Air conditioning and additional facilities
* House prices

## Data Preprocessing & Analysis

The following steps were applied during the project:

* Data exploration and understanding
* Handling missing or inconsistent values
* Encoding categorical features
* Data visualization and correlation analysis
* Feature engineering
* Splitting training and testing data
* Model training and evaluation

## Machine Learning

Regression techniques were used to predict house prices based on property features and analyze the impact of different variables on the final prediction.
